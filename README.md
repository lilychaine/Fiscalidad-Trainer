#  Fiscalidad Trainer — Entrenador Profesional de Impuestos (España)

> Plataforma de entrenamiento tipo test para dominar la fiscalidad española a nivel profesional (máster / oposiciones).

---

##  ¿Qué es esto?

**Fiscalidad Trainer** es una app interactiva que te entrena con preguntas tipo test de alto nivel sobre:

- Impuesto sobre el Valor Añadido  
- Impuesto sobre la Renta de las Personas Físicas  
- Impuesto sobre Sociedades  
- Tributación local  
- Impuesto sobre Transmisiones Patrimoniales y Actos Jurídicos Documentados  
- Impuesto sobre Sucesiones y Donaciones  

Diseñada para:

✔ Estudiantes de máster  
✔ Preparación de oposiciones  
✔ Profesionales que quieren consolidar base técnica  

---

##  Diferencial clave

No es un test cualquiera.

Este sistema incorpora:

- 🔁 **Aprendizaje adaptativo**
- 📊 **Seguimiento de errores**
- 🧠 **Refuerzo de puntos débiles**
- 🎯 **Nivel profesional (no básico)**

---

## 🧪 Modalidades de entrenamiento

### 📘 Estudio por bloque
Entrena por materia y tema específico.

### 📝 Simulacro de examen
20 preguntas mixtas como examen real.

### 🔁 Repaso adaptativo
El sistema detecta tus fallos y te vuelve a entrenar sobre ellos.

---

## 🧠 Cómo funciona

1. Seleccionas modalidad
2. Respondes preguntas tipo test
3. Recibes corrección inmediata
4. El sistema registra errores
5. Se generan refuerzos automáticos

---

## 📊 Qué mide

- Aciertos
- Errores
- % de rendimiento
- Bloques débiles
